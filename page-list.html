Профиль: мой декор

  • /units/add-new-unit
    • Список параметров:
    • 5 фотографий: photo1.*, photo2.*, photo3.*, photot4.*, photo5.*
    • Вес, кг: weight
    • Залог, руб: bail
    • Количество, шт: count
    • Коллекция: set-id
    • Название: title
    • Первый день: first-day-cost
    • Аренда от, дней: rent-min-days
    • Аренда р/сут: day-cost
    • Тип товара: unit-type
    • Массив id цветов: unit-colors
    • id группы: goupid
    • список параметров: [{"id":id1, "value": val1}, {"id":id2, "value": val2}]
    Возможные ответы:
    • OK
    • Текст ошибки
  • /units/get-groups
    • Параметр:
    • parentid: id группы - родителя
    Ответ:
    • json строка, содержащая массив групп. [{"id": 1, "name": "группа №1"}, {"id": 3, "name": "группа X"}]
    • Если нет групп с указанным родителем - возвращается пустой массив ("[]")
  • /units/get-group-parameters
    • Параметр:
    • groupid: id группы, для которой нужны параметры
    Ответ:
    • json массив с параметрами и картинкой: {"picture": "/static/img/imago.jpeh", "parameters": [{"id": id1, "name": имя, "dimension": размерность}, {"id": id2, "name": имя, "dimension": ''}]}
    • если размерность отсутствует - dimension - пустая строка
    • ВАЖНО: parameters содержит json строку (json в json получается)
  • /units/get-my-units
    • Параметры:
    • offset: смещение от начала
    • size: максимальное количество элементов
    • filter: поисковая строка. Пустая строка при отсутствии поиска.
  • /units/color-picker-source
    • Ответ:
    • json словарь следующего содержания:
    • {"group1": [{"id": 1, "rgb_hex": "ABCABA", "texture": "/static/texture1.png"}], "group2": [{"id": 1, "rgb_hex": "ABCABA", "texture": "/static/texture1.png"}]}
    • групп ровно 5
